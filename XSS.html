Cross Site Scripting (XSS)

Stored XSS injection points were found triggering successful execution of javascript code.

Found on Version: 7.13 on October, 2020

Introduction

Opinio is a survey management solution by ObjectPlanet that allows surveys to be designed, published, and managed.

Vulnerability Details

ObjectPlanet Opinio before version 7.13 is vulnerable to stored Cross-Site Scripting (Stored XSS).

Vulnerability Discovery

Stored XSS payload can be saved and later executed.

Payload is injected and saved input field.

The above stored XSS payload is executed when a user visits a page to preview the stored payload.

Results of Exploitation

Arbitrary JavaScript can be executed when accessing pages that displays the stored XSS payload.

Impact

Successful exploitation would allow an attacker to launch client-side attacks by tricking a victim to execute malicious scripts in the victim’s browser. The malicious scripts could potentially allow the attacker to steal sensitive information, logging keystrokes, or perform arbitrary actions on the victim’s behalf. In this scenario, this may allow an attacker to craft and send malicious payloads to a chat agent/chat admin.

Remediation

Apply the latest fix/patch from ObjectPlanet Opinio.
